# Ebook2LateX
Ebook2LaTeX – Công cụ chuyển đổi tự động các tệp ebook (EPUB, MOBI, PDF) sang mã nguồn LaTeX có cấu trúc, hỗ trợ giữ nguyên định dạng tiêu đề, chương, bảng biểu và hình ảnh. Phù hợp cho tác giả, nhà nghiên cứu và nhà xuất bản muốn tái biên soạn nội dung ebook dưới dạng tài liệu LaTeX chuyên nghiệp.
